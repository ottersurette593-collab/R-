# R语言学习教程（从入门到蛋白结构分析项目）

这是一套按能力递进设计的 R 语言学习教程，覆盖：
- 语言介绍
- 安装与配置
- IDE界面与工程化习惯
- 数据类型与基础语法
- 数据处理
- 回归与分类
- 仿真与绘图
- 综合案例
- AI 协作编程
- 结课项目：蛋白结构分析与绘图

## 学习方式建议

1. 先阅读课程地图：`00_course_map/roadmap.md`
2. 按目录顺序学习每一章 `lesson.md`
3. 每章都完成“基础练习 + 进阶练习”
4. 最后独立完成结课项目并输出结果图

## 产出物目标

- 每章至少 1 份练习代码与简短总结
- 结课项目输出：
  - 蛋白结构基础统计表
  - B-factor 趋势图
  - Cα 距离热图（接触图）
  - 方法说明与可复现实验脚本
