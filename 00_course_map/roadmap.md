# 课程地图（Roadmap）

## 总体节奏（建议 8-10 周）

1. 第1-2周：环境搭建 + 基础语法（章节 01-05）
2. 第3-4周：数据处理（章节 06）
3. 第5周：统计建模（章节 07）
4. 第6周：仿真与绘图（章节 08）
5. 第7周：综合案例 + AI 协作（章节 09-10）
6. 第8周：蛋白结构分析大项目（章节 11）

## 递进能力图

- L1 基础操作：会安装、会运行、会读错误信息
- L2 编程能力：可写函数、控制流程、组织脚本
- L3 数据能力：可清洗、变换、汇总、可视化
- L4 建模能力：可做回归与分类并解释结果
- L5 项目能力：可独立完成从数据到报告的完整分析

## 每章固定结构

- 学习目标：知道本章学完会做什么
- 核心讲解：关键概念与最小必要理论
- 课堂示例：可直接运行的示例代码
- 练习任务：基础题 + 进阶题
- 章末检查：自查清单，确保掌握
