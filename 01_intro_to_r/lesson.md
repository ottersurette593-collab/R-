# 01 R语言介绍

## 学习目标

- 了解 R 的定位、优势与应用场景
- 运行第一段 R 代码
- 理解脚本、控制台、包三者关系

## 核心讲解

- R 适用于统计分析、数据科学、可视化、生物信息学
- R 的生态来自 CRAN/Bioconductor 包
- 基本工作流：写脚本 -> 运行 -> 检查输出 -> 修正

## 课堂示例

```r
# Hello R
message("Hello, R!")

# 简单计算
x <- c(1, 2, 3, 4, 5)
mean(x)
sd(x)
```

## 练习任务

1. 基础练习：创建向量 `score <- c(78, 85, 92, 88, 95)`，计算均值和标准差。
2. 进阶练习：写出 3 个你所在领域可以用 R 解决的问题，并给出你预计的输入数据类型。
