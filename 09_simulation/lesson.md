# 09 统计仿真

## 学习目标

- 理解随机模拟与蒙特卡洛方法的基本思想
- 掌握 `set.seed()`、重复抽样、结果汇总流程
- 会用仿真验证统计直觉（如大数定律、中心极限定理）

## 核心讲解

- 仿真流程：定义问题 -> 设定分布/参数 -> 重复模拟 -> 汇总解释
- 常用函数：`rnorm`, `runif`, `rbinom`, `sample`, `replicate`
- 可复现原则：固定随机种子并记录参数

## 课堂示例

```r
set.seed(42)

# 1000 次抛硬币实验，每次抛 100 次，记录正面比例
coin_prop <- replicate(1000, mean(rbinom(100, size = 1, prob = 0.5)))

mean(coin_prop)
sd(coin_prop)
quantile(coin_prop, c(0.025, 0.5, 0.975))
```

## 练习任务

1. 基础练习：改变每次抛硬币次数（20/50/200），比较比例分布离散程度。
2. 进阶练习：用仿真比较三种样本量（30/100/500）下样本均值的稳定性，并写出结论。
