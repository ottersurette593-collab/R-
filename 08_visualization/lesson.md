# 08 数据可视化（先认图，再绘图）

## 学习目标

- 先理解常见图类型及其适用问题
- 掌握图中的基本元素（标题、坐标轴、图例、注释等）
- 再进入 `ggplot2` 的分步绘图实操

## 第一部分：常见图类型

1. 分布图：直方图、密度图、箱线图  
2. 关系图：散点图、拟合线  
3. 比较图：柱状图、分组箱线图  
4. 趋势图：折线图（时间序列）  
5. 组成图：堆叠图（谨慎使用）  

本节提供“图类型示例画廊”，会一次展示上述图形的典型样例，帮助读者先建立视觉直觉。

## 第二部分：图中的基本元素

1. 标题与副标题：告诉读者图在回答什么问题  
2. 坐标轴名称与单位：避免“有数值、没含义”  
3. 图例：解释颜色/形状代表什么  
4. 比例与范围：坐标尺度要合理，不误导  
5. 注释与参考线：突出关键结论  

本节提供“一张曲线图元素说明图”，在同一张图里标注图例、坐标轴、数据曲线、趋势线、参考线等元素。

## 第三部分：具体绘图教学（`ggplot2`）

按“由易到难”练习：

1. 散点图（关系）
2. 散点图 + 回归线（关系趋势）
3. 直方图 + 密度曲线（分布）
4. 箱线图（组间差异）
5. 柱状图（汇总比较）
6. 折线图（时间趋势）
7. 分面图（多组并排比较）
8. 图形导出（`ggsave`, 300dpi）

## 课堂示例

```r
library(ggplot2)

ggplot(mtcars, aes(x = wt, y = mpg, color = factor(cyl))) +
  geom_point(size = 2.5, alpha = 0.9) +
  geom_smooth(method = "lm", se = FALSE) +
  labs(
    title = "车重与油耗关系",
    x = "Weight (1000 lbs)",
    y = "Miles per Gallon",
    color = "Cylinders"
  ) +
  theme_minimal()
```
